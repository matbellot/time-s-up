## Metodo GitHub Pages (gratis e permanente)

### Primo setup (una volta sola):

1. Crea account su [github.com](https://github.com) (gratis)
2. Clicca su "New repository" (pulsante verde)
3. Nome: `times-up-game` (o quello che preferisci)
4. Seleziona "Public"
5. Clicca "Create repository"

### Carica i file:

**Opzione A - Tramite interfaccia web:**
1. Nella pagina del repository, clicca "uploading an existing file"
2. Trascina tutti i file della cartella "Times Up"
3. Scrivi un messaggio (es: "Prima versione")
4. Clicca "Commit changes"

**Opzione B - Tramite GitHub Desktop:**
1. Scarica [GitHub Desktop](https://desktop.github.com)
2. Apri GitHub Desktop e fai login
3. File > Add Local Repository
4. Scegli la cartella "Times Up"
5. Scrivi un messaggio e clicca "Commit to main"
6. Clicca "Publish repository"

### Attiva GitHub Pages:

1. Vai su Settings (nel repository)
2. Scorri fino a "Pages"
3. Sotto "Source", seleziona "main" e "/ (root)"
4. Clicca "Save"
5. Il tuo sito sarÃ  su: `tuousername.github.io/times-up-game`
